# colors

Get the RGB value of one of the known colors.

```sig
light.colors(Colors.Red);
```

## Parameters

* **color**: a well known color, like: ``pink``

## Example

Make all the pixels light up `red`.

```blocks
light.pixels.setAll(light.colors(Colors.Red))
```
