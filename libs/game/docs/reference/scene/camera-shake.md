# camera Shake

Shakes the camera to create an shock effect.

```sig
scene.cameraShake(4, 1000)
```

## Parameters

* **amplitude**: a [number](/types/number) maximum offset the camera will move.
* **duration**: a [number](/types/number) duration in milli-seconds
