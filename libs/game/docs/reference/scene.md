# Scene

Set scene backgrounds and view perspective.

```cards
scene.screenWidth()
scene.screenHeight()
scene.setBackgroundColor(0)
scene.setBackgroundImage(null)
scene.setTileMap(null)
scene.setTile(0, null)
scene.cameraFollowSprite(null)
scene.centerCameraAt(0, 0)
```
