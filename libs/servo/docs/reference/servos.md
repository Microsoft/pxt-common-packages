# Servos

Rotate and run servos connected to the pins.

```cards
servos.P0.setPulse(1500)
servos.P0.setAngle(90)
servos.P0.run(50)
servos.P0.stop()
servos.P0.setRange(0, 180)
servos.P0.setStopOnNeutral(false)
```
